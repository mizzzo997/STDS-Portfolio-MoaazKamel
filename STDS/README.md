# Software 2 Tasks Portfolio

This folder contains all tasks completed for the Software 2 course. Each task includes:

- TaskDescription.md
- Student solution files (Python code, diagrams, or other relevant files)
- Reflection.md

## Tasks Overview

| Task | Description |
|------|-------------|
| Task01_SOLID_Principles | Demonstrates SOLID principles (SRP, OCP, LSP, ISP, DIP) applied to a geometric shapes system. Focuses on maintainability, flexibility, and proper separation of concerns. |
| Task02_Design_Patterns | Implements multiple software design patterns (Factory, Builder, Adapter, Decorator, Observer, Singleton, Strategy) to enhance code reuse, flexibility, and scalability. |
| Task03_Unit_Testing | Applies unit testing, boundary testing, mocking, observer testing, and automation-style testing to ensure correctness and reliability of the shapes system. |
